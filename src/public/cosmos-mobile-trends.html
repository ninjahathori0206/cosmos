✦ 2026 Trends
01
👆
Thumb-First Layout
Primary actions in the bottom 40%. Bottom tab bar navigation. One-hand reachable.
02
🎯
One Accent Colour
Orange for StorePilot, Blue for Foundry. One CTA per screen. All else neutral.
03
☰
Off-Canvas Sidebar
Hamburger → slide-in overlay. Auto-closes on navigation. Dark 40% backdrop.
04
📐
3-Size Type Scale
Display 20px · Body 13–14px · Caption 11px. Mono for codes & numbers.
05
◼
Dark-First Design
Navy dark base (#0F172A). High-contrast accents. Reduces eye strain.
06
💊
Pill Tab Bar
Floating capsule at bottom. 4 tabs max. Active = solid fill. Inactive = ghost.

▸ Key Design Decisions Implemented
Navigation
Off-Canvas Sidebar
Hamburger in topbar → slide-in nav panel with dark backdrop. Auto-closes when a nav item is tapped. No manual close needed.
✓ Live in both apps
Grid Layout
Fluid Grid Collapse
4-col stat grids → 2-col on mobile. fg2/fg3/fg4 form grids, two-col, three-col, main-side all collapse to single column at 768px breakpoint.
✓ Live in both apps
Touch UX
44pt+ Tap Targets
Nav items min-height 48px. Buttons 40px. Qty inputs 72px wide. Pipeline bar horizontally scrollable. Inputs 40px min-height.
✓ Live in both apps
Data & Cards
Smart Scroll + Cards
Tables wrapped in horizontal scroll. SKU cards auto-fill 1fr on mobile. Sticky cart bar full-width. Rate strip stacks vertically.
✓ Live in both apps
